## Configuring bots

### Telegram
To define bot for telegram you could:
1. Define via application.properties (.yml)
   ```properties
   spring.chatbots.telegram.bot.token=1111111111:thisisthetokenforthebot
   spring.chatbots.telegram.bot.name=my_demo_bot_name
   ```
